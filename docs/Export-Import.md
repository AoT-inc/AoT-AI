페이지\: `추가기능 -> 내보내기 가져오기`

선택한 날짜/시간 범위 내에 있는 측정값은 해당 타임스탬프와 함께 CSV로 내보낼 수 있습니다.

또한, 전체 측정 데이터베이스(influxdb)를 ZIP 아카이브 백업으로 내보낼 수 있습니다. 이 ZIP 파일은 측정값을 복원하기 위해 어떤 AoT 시스템에서도 다시 가져올 수 있습니다.

!!! 참고
    측정값은 특정 시스템의 입력/출력 등과 연결된 특정 ID와 연관되어 있습니다. 측정값을 가져오면서 관련된 입력/출력 등을 가져오지 않으면 이러한 측정값을 볼 수 없습니다(예: 대시보드 그래프에서). 따라서 나중에 가져올 때 측정값과 연결된 장치가 가져오는 시스템에서 사용 가능하도록 측정값과 설정을 동시에 내보내는 것이 좋습니다.

!!! 참고
    측정 데이터 가져오기는 기존 데이터를 삭제하지 않으며 현재 측정 데이터에 추가됩니다.

AoT 설정은 AoT 설정 데이터베이스(sqlite)와 사용자 정의 입력, 출력, 함수, 위젯을 포함하는 ZIP 파일로 내보낼 수 있습니다. 이 ZIP 파일은 AoT 및 데이터베이스 버전이 가져오는 시스템의 버전보다 같거나 낮은 경우 다른 AoT 설치로 복원하는 데 사용할 수 있습니다. 또한, 동일한 주요 버전 번호(버전 형식 x.x.x의 첫 번째 숫자)를 가진 시스템에만 가져올 수 있습니다. 예를 들어, AoT 8.5.0에서 설정을 내보내고 AoT 8.8.0에 가져올 수 있지만, AoT 8.2.0(동일한 주요 버전 번호의 이전 버전), 7.0.0(동일한 주요 버전 번호가 아님), 또는 9.0.0(동일한 주요 버전 번호가 아님)에는 가져올 수 없습니다.

!!! 경고
    가져오기는 현재 설정 및 사용자 정의 컨트롤러 데이터를 덮어씁니다(즉, 삭제합니다). 가져오기를 시도하기 전에 AoT 백업을 만드는 것이 권장됩니다.
