경고(Alerts)는 시스템 상태에 대해 사용자에게 알림을 보낼 수 있도록 해줍니다. 예를 들어 센서 모니터링의 경우, 특정 임계값을 초과했을 때 사용자의 주의가 필요한 상황일 수 있습니다.

AoT에는 이메일 알림 기능이 여러 곳에 기본 내장되어 있으며, 입력(Inputs), 출력(Outputs), 제어기(Controllers) 등 여러 위치에서 사용자 정의 Python 코드를 사용할 수 있도록 지원하여 훨씬 다양한 알림 방식을 구현할 수 있습니다.

경고 설정에 대한 자세한 내용은 [경고 설정](Configuration-Settings.md#alert-settings)을 참고하세요.
