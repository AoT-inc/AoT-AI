## AoT Umweltüberwachungs- und Regelsystem

AoT ist Open-Source-Software für den [Raspberry Pi](https://en.wikipedia.org/wiki/Raspberry_Pi). Es verbindet Eingänge und Ausgänge zur Umweltsteuerung.

### Information

See the [README](https://github.com/aot-inc/AoT#uses) for features, projects using AoT, screenshots, and other information.

### Prerequisites

*   Single-board computer (Recommended: [Raspberry Pi](https://www.raspberrypi.org/), any version: Zero, 1, 2, 3, or 4)
*   Debian-based operating system
*   Eine aktive Internetverbindung

### Install

Once booted and logged in, run the following command to initiate the AoT install:

```bash
curl -L https://aot-inc.github.io/AoT/install | bash
```

Öffnen Sie nach der Installation die IP-Adresse des SBC im Browser; Sie werden aufgefordert, einen Admin-Benutzer zu erstellen.

```
https://127.0.0.1
```
