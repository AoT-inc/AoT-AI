페이지\: `설정 -> 메서드`

메서드는 PID에서 설정값 추적(Setpoint Tracking) 및 타이머에서 시간 기반 듀티 사이클 변경을 가능하게 합니다. 일반적으로 PID 컨트롤러는 특정 설정값에 환경 조건을 조정합니다. 설정값이 시간에 따라 변경되도록 하려면 이를 설정값 추적이라고 합니다. 설정값 추적은 리플로우 오븐, 열 순환기(DNA 복제), 자연 일주기 모방 등과 같은 응용 프로그램에 유용합니다. 메서드는 또한 Run PWM Method Conditional과 함께 사용할 때 듀티 사이클을 시간에 따라 변경하는 데 사용할 수 있습니다.

## 시간/날짜 메서드

시간/날짜 메서드는 특정 시간/날짜 범위를 설정값으로 지정할 수 있습니다. 이는 며칠, 몇 주 또는 몇 달에 걸쳐 진행되는 장기 실행 메서드에 유용합니다.

## 지속 시간 메서드

지속 시간 메서드는 특정 시간 지속 후 **설정값**(PID의 경우) 또는 **듀티 사이클**(조건의 경우)을 설정할 수 있습니다. 추가된 각 새로운 지속 시간은 이전 지속 시간 뒤에 쌓이므로, 새로 추가된 **시작 설정값**은 이전 항목의 **종료 설정값** 이후에 시작됩니다.

"메서드 반복" 옵션을 사용하면 메서드가 끝에 도달했을 때 반복됩니다. 이 옵션이 사용되면 메서드에 더 이상 지속 시간을 추가할 수 없습니다. 반복 옵션이 삭제되면 지속 시간을 더 추가할 수 있습니다. 예를 들어, 메서드가 총 200초이고 반복 지속 시간이 600초로 설정된 경우, 메서드는 3번 반복된 후 PID 또는 조건을 자동으로 종료합니다.

## 일일(시간 기반) 메서드

일일 시간 기반 메서드는 시간/날짜 메서드와 유사하지만 매일 반복됩니다. 따라서 이 메서드에서는 하루의 범위만 설정해야 합니다. 시작 시간을 00:00:00으로 설정하고 종료 시간을 23:59:59(또는 00:00:00, 즉 시작 시간으로부터 24시간)으로 설정하십시오. 시작 시간은 이전 종료 시간보다 크거나 같아야 합니다.

## 일일(사인파) 메서드

일일 사인파 메서드는 사인파를 기반으로 하루 동안의 설정값을 정의합니다. 사인파는 y = [A \* sin(B \* x + C)] + D로 정의되며, 여기서 A는 진폭, B는 주파수, C는 각도 이동, D는 y축 이동입니다. 이 메서드는 매일 반복됩니다.

## 일일(베지어 곡선) 메서드

일일 베지어 곡선 메서드는 3차 베지어 곡선을 기반으로 하루 동안의 설정값을 정의합니다. 베지어 곡선에 익숙하지 않은 경우 [그래픽 베지어 곡선 생성기](https://www.desmos.com/calculator/cahqdxeshd)를 사용하여 4개의 점(각각 x와 y 세트)에 대한 8개의 변수를 생성하는 것이 좋습니다. x축 시작(x3)과 종료(x0)는 24시간 내에 자동으로 확장 또는 왜곡되며 이 메서드는 매일 반복됩니다.

## 캐스케이드 메서드

이 메서드는 여러 메서드를 결합하고 메서드의 평균을 출력합니다. 예를 들어, 60초 동안 100으로 설정된 지속 시간 메서드와 60초 동안 0으로 설정된 지속 시간 메서드(영원히 반복되도록 설정됨)를 00:00:00에 0에서 50으로 상승하고 12:00:00에 다시 0으로 하락하는 일일 메서드와 결합한다고 가정합니다. 00:00:00에 결합된 메서드는 60초마다 0((0 / 100) \* (0 / 100) = 0)에서 0((100 / 100) \* (0 / 100) = 0)으로 진동하며, 점차 증가하여 12:00:00에는 60초마다 0((0 / 100) \* (50 / 100))에서 50((100 / 100) \* (50 / 100))으로 진동합니다. 이는 간단한 예제이지만, 조합은 매우 복잡해질 수 있습니다.
